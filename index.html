Link to NuBitsj repository
The NuBitsj library is a Java implementation of the Nubits protocol, which allows it to maintain a wallet and send/receive transactions without needing to do full blockchain validation. It comes with full documentation and some example apps showing how to use it. NuBitsj uses a central server to download block hashes for verification.

Link to LiquidBits.net repository
In a nutshell: You run a program that places NBT, EUR, USD and BTC orders on an exchange with a very small spread. The program will send proofs of those orders to a server which in turn pays you interest in the form of NuBits.
You can find the instructions to participate here.
You can find the awesome Plunge repository here which provides a nice graphical interface.

Link to Abe NuBits Explorer repository
This software reads the NuBits blockfile, transforms and loads the data into a database, and presents a web interface similar to Bitcoin Block Explorer.
A working installation showing the NuBits blockchain can be found here.
